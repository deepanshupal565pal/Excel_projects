# 📊 Puma Sales Dashboard

## Overview
This repository contains the **Puma Sales Dashboard** built in Microsoft Excel.  
The dashboard provides insights into Puma’s sales performance, customer trends, and product analysis in a visual and interactive format.

---

## 📂 Repository Contents
- `Puma sales dashboard.xlsx` → The main Excel dashboard file
- `README.md` → Project documentation
